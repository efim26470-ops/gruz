# KartuliGo — самоучитель грузинского для GitHub Pages

Отдельное HTML/PWA-приложение для изучения грузинского языка: собственное название, собственный `localStorage`, отдельный `manifest`, отдельный service worker и отдельные иконки.

## Что есть внутри

- уровни A0 → A1 → A2 → A2+ → B1 → B2;
- грузинский алфавит мхедрули: 33 буквы;
- карточки для запоминания слов;
- упражнения: выбор перевода, выбор грузинского слова, аудирование, набор ответа, пары, сбор фразы;
- режим `ქართული / Translit`;
- встроенная мини-клавиатура с грузинскими буквами;
- произношение через Web Speech API с языковой меткой `ka-GE`;
- прогресс, XP, серия дней и сложные карточки в `localStorage`;
- адаптация под iPhone и PWA-иконка.

## Публикация на GitHub Pages

1. Создай новый репозиторий, например `kartuligo`.
2. Загрузи в корень репозитория файлы из этой папки: `index.html`, `styles.css`, `app.js`, `manifest.webmanifest`, `sw.js`, `icons/`.
3. Открой `Settings → Pages`.
4. Выбери `Deploy from a branch`, ветка `main`, папка `/root`.
5. После публикации открой URL на iPhone в Safari и добавь сайт на экран «Домой».

## Важно про озвучку

Озвучка использует голоса, которые доступны в браузере/системе пользователя. Если грузинский голос не установлен, браузер может читать хуже или использовать fallback-голос. Логика приложения при этом остаётся рабочей.
