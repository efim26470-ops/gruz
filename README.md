# KartuliGo — самоучитель грузинского для GitHub Pages

Отдельное HTML/PWA-приложение для изучения грузинского языка: собственное название, собственный `localStorage`, отдельный `manifest`, отдельный service worker и отдельные иконки.

## Что есть внутри

- уровни A0 → A1 → A2 → A2+ → B1 → B2;
- грузинский алфавит мхедрули: 33 буквы;
- карточки для запоминания слов;
- упражнения: выбор перевода, выбор грузинского слова, аудирование, набор ответа, пары, сбор фразы;
- режим `ქართული / Tr`: точная опорная транскрипция с kʼ, pʼ, tʼ, qʼ, tsʼ, chʼ;
- встроенная мини-клавиатура с грузинскими буквами;
- исправленная озвучка: Web Speech получает именно грузинский текст с `ka-GE`, без чтения английского транслита;
- прогресс, XP, серия дней и сложные карточки в `localStorage`;
- адаптация под iPhone и PWA-иконка.

## Публикация на GitHub Pages

1. Создай новый репозиторий, например `kartuligo`.
2. Загрузи в корень репозитория файлы из этой папки: `index.html`, `styles.css`, `app.js`, `manifest.webmanifest`, `sw.js`, `icons/`.
3. Открой `Settings → Pages`.
4. Выбери `Deploy from a branch`, ветка `main`, папка `/root`.
5. После публикации открой URL на iPhone в Safari и добавь сайт на экран «Домой».

## Важно про озвучку

Озвучка использует голоса, доступные в браузере/системе пользователя. Приложение больше не подменяет грузинский текст английским транслитом: при нажатии 🔊 в Speech API отправляется грузинская строка с языковой меткой `ka-GE`. Если система не имеет грузинского голоса, браузер может произнести хуже или вернуть ошибку — в этом случае нужно установить Georgian voice/keyboard или попробовать Safari/Chrome.


## Изменения в этой версии

- Расширен словарь: теперь 328 карточек вместо исходных 146.
- Транскрипция пересчитана по грузинским буквам: различаются k/kʼ, p/pʼ, t/tʼ, qʼ, ts/tsʼ, ch/chʼ.
- Локальный прогресс сохранён в прежнем ключе `kartuligo.progress.v1`, поэтому обновление не сбрасывает XP/уроки.
- Service Worker обновлён до `kartuligo-cache-v2`, чтобы GitHub Pages быстрее подтянул новые файлы.
