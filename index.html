Установка на iPhone

Открой опубликованный сайт в Safari, нажми Поделиться → На экран «Домой» → Добавить. После этого KartuliGo будет открываться как отдельное приложение.

  1. Загрузи файлы в отдельный репозиторий GitHub.
  2. Включи GitHub Pages для ветки main.
  3. Открой URL на iPhone в Safari и добавь на экран «Домой».

Озвучка работает через Web Speech API. Озвучка больше не читает английский транслит: приложение отправляет в Web Speech именно грузинский текст с меткой ka-GE. Для лучшего результата установи грузинский системный голос, если он доступен на устройстве.
